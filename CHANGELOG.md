# Changelog

## 1.0.0

First stable local-first Android release.

- Offline SQLite storage with Schema 4 migrations.
- Beer and Tasting create, edit, detail, search, filter, sort, soft-delete, and restore flows.
- Custom countries, categories, styles, flavor tags, five-option sensory ratings, and local multi-photo support.
- Local image compression, cover photos, photo recovery, statistics, and JSON backup/restore.
- Android App Shell, safe-area layout, overlay navigation, and back-button handling.

Known limitations: no cloud sync, accounts, multi-device sharing, multi-language mode, or dark mode. JSON backup should be exported before uninstalling or clearing app data.
