FAQ

常见问题

Beer Journal 会同步到云端吗？

不会。1.0 完全离线，没有账号、服务器同步或多设备共享。

照片和数据存在哪里？

记录保存在本地 SQLite，压缩照片保存在 Android App 私有目录，不会作为公开链接上传。

卸载后还能找回数据吗？

不能保证。卸载或清除数据会移除本地内容，请先导出 JSON 备份。

支持 iPhone 吗？

当前版本是 Android APK。其他平台暂不在 1.0 范围内。
